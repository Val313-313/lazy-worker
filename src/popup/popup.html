Was ist Lazy Worker?

Lazy Worker erfasst automatisch deine Job-Bewerbungen und hilft dir, das RAV-Formular "Nachweis der persoenlichen Arbeitsbemuehungen" auf arbeit.swiss auszufuellen.

Bewerbung erfassen

1. Oeffne ein Job-Inserat auf einer der unterstuetzten Plattformen

2. Klicke auf "Seite erfassen"

3. Firma, Stelle und Ort werden automatisch erkannt

Unterstuetzte Plattformen

jobs.ch jobup.ch LinkedIn Indeed job-room.ch

Andere Seiten werden ebenfalls unterstuetzt, die Erkennung kann aber weniger genau sein.

Bearbeiten & Loeschen

Klicke auf eine Bewerbung, um sie zu bearbeiten. Mit dem X auf der Karte kannst du sie direkt loeschen.

RAV-Formular ausfuellen

1. Gehe auf arbeit.swiss und oeffne das Nachweisformular

2. Der Lazy Worker Button erscheint unten rechts

3. Waehle eine Bewerbung aus und die Felder werden automatisch ausgefuellt

CSV Export

Mit dem Download-Button oben rechts kannst du alle Bewerbungen als CSV-Datei exportieren.

Datenschutz

Alle Daten werden lokal in deinem Browser gespeichert. Es werden keine Daten an externe Server gesendet.
